<h1>Installing</h1>

Firstly you have to install requirements for using bot

<pre lang="bash"> pip install -r requirements.txt </pre>
<br>

<h1>Configuration</h1>

Set bot token to your and to your user id in data.py

<pre lang='python'> 
bot_settings = {'token': 'YOUR-BOT-TOKEN'}

tg_id = YOUR USER ID
</pre>
